# Dashboard Financeiro 

## Visão Geral

Dashboard financeiro desenvolvido para análise estratégica de receitas, despesas e fluxo de caixa, com foco em clareza visual, tomada de decisão e organização financeira.

O sistema apresenta indicadores-chave, gráficos dinâmicos e detalhamento completo das transações, permitindo uma leitura executiva e operacional dos dados.

---

## Funcionalidades

* Indicadores principais (Receita, Despesa, Resultado, Margem)
* Visualização de fluxo financeiro (recebido vs pago)
* Classificação por status:

  * Recebido / Quitado
  * Em aberto
  * Atrasado
* Gráficos interativos com Chart.js
* Distribuição por categorias
* Barras de progresso por desempenho
* Tabelas detalhadas de receitas e despesas
* Insight automático baseado nos dados

---

## Tecnologias Utilizadas

* HTML5
* CSS3 (Design System customizado)
* JavaScript (ES6+)
* Chart.js (visualização de dados)
